# SiteSpector — Marketing Starter Pack

**Dla zespołu:** Aleksandra, Daniel (+ 3. osoba)  
**Cel:** Szybkie wejście w tematykę, spójna komunikacja i gotowe narzędzia do pracy.

Starter Pack powstał na bazie zaktualizowanych dokumentów strategicznych SiteSpector (wizja, odbiorcy, kanały, ToV, ścieżka, cennik) oraz ustaleń z transkrypcji spotkań. To **jedno źródło prawdy** do kreacji, postów i kampanii.

## Zawartość

| Plik | Opis |
|------|------|
| [01-meta-ads-wytyczne.md](01-meta-ads-wytyczne.md) | Budżet, grupy, testy A/B, piksel, Lookalike, remarketing |
| [02-tone-of-voice.md](02-tone-of-voice.md) | Ton marki, zasady pisania, ton per kanał (LinkedIn, Meta Ads) |
| [03-prompty-ai.md](03-prompty-ai.md) | Gotowe prompty do ChatGPT/Cursor pod posty, reklamy, opisy |
| [04-priorytety-grup-docelowych.md](04-priorytety-grup-docelowych.md) | 7 segmentów, persony, kto w pierwszej fazie, targety pod Meta |

## Jak z tego korzystać

1. **Przed kreacjami** — przejrzyj [02-tone-of-voice.md](02-tone-of-voice.md) i odpowiedni segment w [04-priorytety-grup-docelowych.md](04-priorytety-grup-docelowych.md).
2. **Meta Ads** — ustawienia i budżety zawsze według [01-meta-ads-wytyczne.md](01-meta-ads-wytyczne.md).
3. **Drafty postów / reklam** — start od promptu z [03-prompty-ai.md](03-prompty-ai.md), potem dopasowanie do ToV.

Pełna dokumentacja produktu i strategii pozostaje w repozytorium (docs-new, CHANGELOG). Ten folder to **wyciąg pod marketing**.
