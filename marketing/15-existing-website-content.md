## 2. Strona Funkcji (/funkcje)

### 10 modułów z opisami

```
1. Crawling SEO
   Profesjonalny crawl z Screaming Frog Engine — tytuły, meta, H1, 
   linki, obrazy, kody odpowiedzi, przekierowania...
   
2. Wydajność (Lighthouse)
   Core Web Vitals desktop i mobile — LCP, CLS, TBT, TTFB, FCP, 
   Speed Index, accessibility, best practices...
   
3. Widoczność (Senuto)
   Pozycje organiczne, słowa kluczowe, wzrosty/straty, sezonowość,
   domeny konkurentów, kanibalizacja...
   
4. AI Overviews
   Monitoring widoczności w odpowiedziach AI Google, słowa kluczowe
   gdzie jesteś cytowany, konkurenci w AIO...
   
5. Backlinki (Senuto)
   Statystyki backlinków, referring domains, anchory, atrybuty linków...
   
6. Analiza AI (Gemini)
   9 kontekstowych analiz, strategia cross-tool, roadmap, executive summary,
   quick wins...
   
7. Execution Plan
   Do 200 priorytetyzowanych zadań z kodem, 8 modułów, quick wins...
   
8. Raporty PDF
   Executive (15-25 str), Standard (50-80 str), Full (80-150+ str)...
   
9. Zespoły i Workspace
   Role (owner, admin, member), multi-workspace, projekty...
   
10. Harmonogramy
    Automatyczne audyty daily/weekly/monthly...
```

**Uwagi:**
- Ta strona wymaga rozbudowania — każdy moduł powinien mieć 2-3 zdania opisu i benefit dla użytkownika
- Brakuje sekcji "Jak to porównuje się z osobnymi narzędziami?"

## 9. Tooltips i Definicje w App (lib/tooltips.ts)

Narzędzie ma wbudowane opisy metryk — użyteczne przy tworzeniu materiałów edukacyjnych:

| Metryka | Definicja |
|---------|-----------|
| Wynik Ogólny | Agregat wszystkich wyników (SEO, performance, content) |
| Wynik SEO | Techniczne SEO na podstawie crawla i Lighthouse |
| LCP | Largest Contentful Paint — czas do wyrenderowania głównego elementu |
| FCP | First Contentful Paint — czas do pojawienia pierwszych treści |
| CLS | Cumulative Layout Shift — stabilność układu |
| TBT | Total Blocking Time — czas blokowania interakcji |
| TTFB | Time to First Byte — czas odpowiedzi serwera |
| Speed Index | Jak szybko widać treść |
