# SiteSpector - Professional Website Audit Platform

AI-powered SaaS platform for comprehensive website audits. SEO analysis, performance metrics, content quality, and competitor comparison with AI recommendations.

## 🚀 Quick Links
- **Public Documentation**: [docs/README.md](docs/README.md)
- **Technical Documentation**: [.context7/INDEX.md](.context7/INDEX.md)
- **Agent Onboarding**: [agents.md](agents.md)

## 🎯 Core Features
- **SEO Crawling**: Technical analysis via Screaming Frog.
- **Performance**: Web Vitals analysis via Lighthouse.
- **AI Analysis**: Content quality via Google Gemini.
- **Multi-tenancy**: Team workspaces and role-based access.
- **Billing**: Integrated Stripe subscriptions.

## 🛠️ Development
SiteSpector uses a **VPS-only Docker workflow**. Code locally in Cursor and deploy to the Hetzner VPS via SSH.
