Analiza tagów Title

Tag <title> to jeden z najsilniejszych czynników rankingowych. Wyświetla się jako niebieski link w wynikach Google. Optymalna długość to 50–60 znaków. Każda strona powinna mieć unikalny, zawierający słowo kluczowe tytuł.

Analiza Meta Description

Meta description nie jest bezpośrednim czynnikiem rankingowym, ale wpływa na CTR — dobrze napisany opis zachęca do kliknięcia i może zwiększyć ruch organiczny o 5–30%. Optymalna długość to 70–160 znaków. Modele AI (ChatGPT, Perplexity) używają meta description do oceny trafności strony.

{{ kv_row('Strony z meta desc', onpage.has_desc_count ~ ' / ' ~ onpage.total_indexable) }} {{ kv_row('Brak meta description', onpage.missing_descriptions, 'badge-warning' if onpage.missing_descriptions > 0 else '') }} {{ kv_row('Duplikaty meta desc', onpage.duplicate_descriptions, 'badge-warning' if onpage.duplicate_descriptions > 0 else '') }} {{ kv_row('Zbyt krótkie (<70 zn.)', onpage.short_descriptions) }} {{ kv_row('Zbyt długie (>160 zn.)', onpage.long_descriptions) }}
Optymalne parametry meta desc:
  • Długość: 70–160 znaków
  • Zawiera główne słowo kluczowe
  • Zawiera CTA (np. "Sprawdź", "Dowiedz się")
  • Unikalny i opisuje korzyść dla użytkownika

Analiza nagłówków H1

Nagłówek H1 to najważniejszy nagłówek na stronie — powinien być dokładnie jeden, unikalny, zawierać główne słowo kluczowe i opisywać główny temat strony. Brak H1 lub wiele H1 na stronie to typowy błąd negatywnie wpływający na SEO. Nagłówki H1 nie powinny być identyczne z tagiem <title>.

{{ kv_row('Strony z H1', onpage.has_h1_count ~ ' / ' ~ onpage.total_indexable) }} {{ kv_row('Brak H1', onpage.missing_h1, 'badge-warning' if onpage.missing_h1 > 0 else '') }} {{ kv_row('Duplikaty H1 (grup)', onpage.duplicate_h1_groups, 'badge-warning' if onpage.duplicate_h1_groups > 0 else '') }} {{ kv_row('H1 = Title (identyczne)', onpage.h1_equals_title_count) }} {{ kv_row('Strony z H2', onpage.has_h2_count) }}
Zalecenia dotyczące H1:
  • Dokładnie jeden H1 na podstronie
  • Unikalny w skali całej domeny
  • Różny od tagu <title> (uzupełniający)
  • Zawiera główne słowo kluczowe
  • Nie stosuj H1 w nagłówkach/stopkach (site-wide)

Obrazy bez atrybutu ALT ({{ onpage.images_without_alt }} łącznie)

Atrybut ALT opisuje zawartość obrazu dla robotów wyszukiwarek i czytników ekranu (accessibility). Brak ALT oznacza, że Google nie może zaindeksować obrazu w Google Grafika. Szczególnie ważne dla witryn e-commerce gdzie obrazy produktów mogą generować ruch. Nie wszystkie obrazy wymagają ALT — ikony dekoracyjne i tła powinny mieć pusty ALT (alt="").
