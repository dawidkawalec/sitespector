## Status

API Access jest w przygotowaniu i będzie dostępne w planie **Enterprise**.

## Planowane endpointy

- `POST /api/audits` — tworzenie audytu
- `GET /api/audits` — lista audytów
- `GET /api/audits/{id}` — szczegóły audytu
- `GET /api/audits/{id}/pdf` — pobranie raportu PDF
- `GET /api/audits/{id}/raw` — pobranie surowych danych

## Autentykacja

Planowana autentykacja przez API key (per workspace).
