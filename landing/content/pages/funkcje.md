## Crawling SEO (Screaming Frog)

PLACEHOLDER: Content agent — szczegółowy opis crawlingu:
- Profesjonalny crawl całej strony (Screaming Frog SEO Spider)
- Analiza meta tytułów i opisów (obecność, długość, duplikaty)
- Nagłówki H1-H6 (struktura, duplikaty, brak H1)
- Obrazy (ALT tagi, rozmiary)
- Linki wewnętrzne (struktura, broken links, orphan pages)
- Status codes (200, 301, 302, 404, 500)
- Canonicale i dyrektywy (robots.txt, meta robots)
- Hreflang (dla stron wielojęzycznych)
- Sitemap detection
- Indexability status

## Analiza wydajności (Google Lighthouse)

PLACEHOLDER: Content agent — szczegółowy opis Lighthouse:
- Audyt desktop I mobile w jednym przebiegu
- Core Web Vitals: LCP, FCP, CLS, TTFB, TBT, Speed Index
- Performance Score (0-100)
- Accessibility Score (0-100)
- Best Practices Score (0-100)
- SEO Score (0-100)
- 176 szczegółowych audytów: diagnostyki, opportunities, passed
- Konkretne rekomendacje optymalizacji z szacowanym wpływem

## Widoczność w Google (Senuto)

PLACEHOLDER: Content agent — szczegółowy opis integracji Senuto:
- Dashboard widoczności domeny
- Pozycje słów kluczowych (TOP3, TOP10, TOP50) z paginacją do 10,000
- Trendy widoczności (wykresy sezonowości)
- Wins/Losses — które frazy zyskujesz, które tracisz
- Kanibalizacja — które strony konkurują o te same frazy
- Sekcje — analiza subdomen i katalogów
- AI Overviews — statystyki, słowa kluczowe z intencjami, porównanie z konkurencją
- Profil backlinków — statystyki, anchory, domeny odsyłające
- Konfigurowalny kraj analizy (domyślnie Polska)

## Analiza AI (Google Gemini)

PLACEHOLDER: Content agent — szczegółowy opis AI:
- Google Gemini 3.0 Flash — najnowsza generacja
- Analiza jakości treści (score 0-100, czytelność, ton)
- Kontekstowe analizy per obszar: SEO, performance, visibility, AI Overviews, backlinks, links, images
- Cross-tool korelacje — połączenia między narzędziami
- Roadmapa działań: natychmiastowe → krótkoterminowe → średnio → długoterminowe
- Quick Wins — zunifikowana lista z wszystkich analiz
- Executive Summary — zdrowie strony, mocne strony, krytyczne problemy
- ROI Action Plan z oceną impact/effort
- Sugestie poprawek per problem
- Generowanie ALT tagów dla obrazów
- Deep analysis per strona

## Raporty PDF

PLACEHOLDER: Content agent — opis raportów PDF:
- Automatyczne generowanie (WeasyPrint + Jinja2)
- 9 profesjonalnych sekcji:
  1. Executive Summary
  2. Spis treści (dynamiczny)
  3. Analiza SEO Techniczna
  4. Analiza Wydajności
  5. Analiza Treści
  6. Local SEO (jeśli dotyczy)
  7. Analiza Konkurencji (jeśli dostępna)
  8. Action Plan
  9. Załączniki
- Kolorowe rekomendacje (zielone, żółte, czerwone, niebieskie AI)
- White-label w Pro/Enterprise
- Eksport surowych danych (ZIP z JSON)
