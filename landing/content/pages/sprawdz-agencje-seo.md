## Płacisz za SEO, ale nie widzisz efektów?

PLACEHOLDER: Content agent — napisz sekcję opisującą typowe problemy właścicieli firm ze swoimi agencjami SEO:
- Płacisz X tysięcy miesięcznie, ale nie wiesz co agencja robi
- Dostajesz raporty, ale nie rozumiesz ich
- Pozycje nie rosną, ruch się nie zmienia
- Agencja mówi "SEO wymaga czasu" ale minął rok

## Jak zweryfikować agencję w 3 krokach

PLACEHOLDER: Content agent — opisz 3 proste kroki:
1. Wpisz URL swojego sklepu w SiteSpector (darmowy plan — 5 audytów)
2. Poczekaj 1-3 minuty na pełny audyt (SEO + wydajność + widoczność + AI)
3. Porównaj wyniki z tym, co mówi Twoja agencja

Pokaż co SiteSpector sprawdzi automatycznie:
- Meta tagi i nagłówki (Screaming Frog) — czy agencja je poprawiła?
- Core Web Vitals (Lighthouse) — czy strona jest szybka?
- Widoczność w Google (Senuto) — czy pozycje rosną?
- AI Overviews — czy Twoja strona pojawia się w AI wyszukiwania?
- Profil backlinków — czy agencja buduje linki?

## Na co zwrócić uwagę w audycie

PLACEHOLDER: Content agent — lista konkretnych rzeczy do sprawdzenia:
- Performance Score poniżej 50 = problem z szybkością
- Brakujące meta opisy i tytuły = podstawowe zaniedbanie
- Brak H1 lub zduplikowane H1 = źle zoptymalizowana struktura
- Obrazy bez ALT tagów = niewykorzystany potencjał SEO
- Widoczność w Senuto — porównaj trend z datą rozpoczęcia współpracy z agencją
- Quick Wins — ile łatwych poprawek agencja mogła już dawno wdrożyć?

## Czerwone flagi — kiedy zmienić agencję

PLACEHOLDER: Content agent — opisz sygnały ostrzegawcze:
- Performance Score nie rośnie po 3 miesiącach
- Agencja nie potrafi wyjaśnić co robi
- Widoczność w Senuto spada mimo "optymalizacji"
- Agencja nie dostarcza raportów z konkretnymi danymi
- Te same błędy techniczne pojawiają się co miesiąc
- Brak nowych backlinków mimo obietnic link buildingu

## Przykładowy raport audytu

PLACEHOLDER: Content agent — opisz co zawiera raport PDF z SiteSpector:
1. Executive Summary — ogólna ocena strony
2. Analiza SEO — crawl, meta tagi, nagłówki, linki
3. Wydajność — Core Web Vitals desktop i mobile
4. Treść — jakość contentu, czytelność, rekomendacje AI
5. Local SEO — jeśli firma lokalna
6. Analiza konkurencji — porównanie z 3 konkurentami
7. Widoczność — pozycje w Google, trendy, AI Overviews
8. Backlinki — profil linków przychodzących
9. Action Plan — priorytetyzowana lista zadań
