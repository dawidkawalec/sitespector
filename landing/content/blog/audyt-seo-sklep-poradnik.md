## Dlaczego audyt sklepu różni się od zwykłej strony

PLACEHOLDER: Content agent — Specyfika e-commerce: duża liczba URL-i, filtry, paginacja, zduplikowane treści, product schema, crawl budget. Wyzwania: canonicaly, noindex, thin content. 2–3 akapity.

## Krok 1: Crawling i indeksacja

PLACEHOLDER: Content agent — Co sprawdzać: status codes (200, 301, 404, 500), canonicaly, robots.txt, sitemap, parametery URL (Google Search Console). Problemy typowe dla sklepów: parametry ?sort=, ?color= generujące tysiące wariantów. SiteSpector: raport crawlingowy.

## Krok 2: Core Web Vitals i wydajność

PLACEHOLDER: Content agent — LCP, CLS, TTFB — dlaczego kluczowe dla konwersji i SEO. Strony produktu vs kategorie vs landing pages. SiteSpector: Lighthouse per page type. Konkretne progi: LCP < 2.5s, CLS < 0.1.

## Krok 3: On-page — meta, nagłówki, treść

PLACEHOLDER: Content agent — Unikalne tytuły i opisy dla kategorii/produktów. H1, struktura H2–H6. Thin content: jak rozpoznać, co robić (rozbudowa, łączenie stron). Obrazy: ALT, rozmiar. SiteSpector: sekcja on-page w raporcie.

## Krok 4: Schema i dane strukturalne

PLACEHOLDER: Content agent — Product schema, BreadcrumbList, Organization. Błędy w GSC. Narzędzia: Schema Validator. SiteSpector: czy wspiera schema audit — jeśli tak, opisać; jeśli nie, wspomnieć ogólnie.

## Krok 5: Analiza treści z AI

PLACEHOLDER: Content agent — AI (Gemini) analizuje treść: czy odpowiada intencji, czy jest kompletna, sugestie ulepszeń. Przykłady: opisy kategorii, „O nas”, blog. SiteSpector: AI rekomendacje w raporcie.

## Implementacja — priorytety i harmonogram

PLACEHOLDER: Content agent — Action plan: krytyczne (błędy 5xx, indeksacja) → ważne (CWV, meta) → nice-to-have (schema, treść). Harmonogram: tydzień 1–2, miesiąc 1–3. SiteSpector generuje listę zadań — jak z niej korzystać.

## Checklista audytu sklepu (do wydruku)

PLACEHOLDER: Content agent — Krótka lista 12–15 punktów: crawling OK, CWV w normie, unikalne meta, schema bez błędów, treść nie thin, itd. Format checkbox. CTA: uruchom audyt w SiteSpector.
